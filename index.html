Work

Robingood

Robingood is a Robinhood clone. It Replicates the CRUD functions to create and add/remove stocks to watchlists as well as buying/selling stocks. You are also able to add a bank account, deposit or withdraw funds and set up a recurring investment.

React, Redux, CSS, Python, Flask, SQLAlchemy, JavaScript, PostgreSQL, Polygon API, Chart.js

Petsy

Petsy is an Etsy clone. It Replicates the CRUD functions to post/sell a product as well as adding products to cart for purchase.

React, Redux, CSS, Python, Flask, SQLAlchemy, JavaScript, PostgreSQL

PearBnb

PearBnb is a, AirBnb clone. It Replicates the CRUD functions to create and delete a spot as well as post a review on a spot.

React, Redux, CSS, JavaScript, Express, SQL, JavaScript, PostgreSQL
